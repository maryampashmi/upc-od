Configuration

The Schema contains the definition of all know objects types to which all the sources need to be mapped. For each object type the following properties are defined:

  • a list containing the available fields
  • the conditions under which two objects are identical

On the other side, the configuration of sources defines the available sources. For every source the following properties are defined:

  • A human readable name
  • The name of the adapter used in order to access the source
  • Additional configuration provided to the adapter (in an adapter specific format)
  • The mapping from the source's schema to our internal schema
